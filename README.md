# gulp-build

gulp-build是一个基于gulp的前端工程构建工具。


## 目录结构

```
.
├── .git                # git
├── project                 # 项目文件夹
│   ├── src              # 图片
│   |    ├── images              # 图片
│   |    ├── index.html          # 项目入口html
│   |    ├── scripts             # 脚本
│   |    ├── styles              # 样式
│   |    ├── tests               # 测试
│   |    └── views               # 视图
|   ├── dist
|   └── dev  
├── assets              # 资源
│   ├── api.json            # server api 配置
│   └── data                # mock api 数据
├── common               # common模块
├── gulpfile.js         # gulp脚本入口
├── .gitignore          # git忽略配置
└── package.json        # 包配置文件

```
